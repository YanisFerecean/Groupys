# mobile/components/feed/

Social feed components.

## Files

| File | Purpose |
|---|---|
| `FeedPostCard.tsx` | Full post card in feed: author info, content, media, reactions, comments count |
| `FeaturedPost.tsx` | Large featured/editorial post card |
| `ArtistSpotlight.tsx` | Artist spotlight card in feed sidebar |
| `PostGrid.tsx` | Grid layout of post thumbnails (for profile/community views) |
| `PostListItem.tsx` | Compact post list item variant |
