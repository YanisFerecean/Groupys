# mobile/components/match/

User matching and dating components.

## Files

### `UserRecommendationCard.tsx`
Swipeable user card for discovery. Shows user photo, name, compatibility score, shared genres/artists. Supports gesture-based swipe left (pass) / right (like).

### `MatchCard.tsx`
Smaller match card variant used in match history grid.

### `ActionButtons.tsx`
Like (heart) and Pass (X) action buttons below the card stack.

### `MatchCelebrationModal.tsx`
Animated celebration modal when a mutual match occurs. Shows both users and a "Start chatting" CTA.

### `InsightRow.tsx`
Row showing a match insight — shared artist, genre, or listening pattern.

### `MatchHistoryListItem.tsx`
List item for past matches with user info and matched date.

### `MatchListItem.tsx`
Compact list item variant for match lists.

### `SentLikeListItem.tsx`
List item for pending outgoing likes (awaiting response).
