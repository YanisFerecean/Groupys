# mobile/components/profile/

User profile display and editing components.

## Files

### `ProfileHeader.tsx`
Profile banner, avatar, display name, bio, country, tags, follower/following counts. Adapts for own vs public profile.

### `PublicProfileScreen.tsx`
Read-only profile screen for other users. Shows follow/DM buttons, widget grid.

### `EditProfileModal.tsx`
Full-screen modal for editing profile: display name, bio, country, tags, banner, widget configuration.

### `SpotifyConnectButton.tsx`
Button to initiate Spotify OAuth flow. Shows connected/disconnected state.

### `ListeningProfile.tsx`
Music taste summary section: top genres, top artists, listening stats.

### `TopArtists.tsx`
Grid of user's top artists with images.

### `AlbumOfWeek.tsx`
Featured album display widget.

### `CountryPicker.tsx`
Country selection picker for profile editing.

### `GenrePicker.tsx`
Genre selection picker (multi-select) for profile preferences.
