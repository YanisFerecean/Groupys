# mobile/components/community/

Community display and management components.

## Files

### `CommunityDetailScreen.tsx`
Full community detail screen: banner, description, member count, join/leave button, post feed, member list. Used across all tab stacks.

### `CreateCommunityModal.tsx`
Modal form for creating a new community: name, description, genre, icon (emoji or image), tags.

### `EditCommunityModal.tsx`
Modal for editing community details (owner only). Same fields as create modal.
