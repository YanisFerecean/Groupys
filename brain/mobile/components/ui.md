# mobile/components/ui/

Reusable primitive UI components (mobile equivalent of web's shadcn/ui).

## Files

| File | Purpose |
|---|---|
| `AuthImage.tsx` | Renders images with Clerk auth token in headers |
| `AuthImageWithToken.tsx` | Variant that accepts token prop directly (for list items) |
| `Avatar.tsx` | Circular user avatar with fallback initials |
| `Badge.tsx` | Small colored label/tag |
| `Button.tsx` | Styled touchable button with variants |
| `Card.tsx` | Rounded card container with shadow |
| `FullscreenSpinner.tsx` | Full-screen loading indicator overlay |
| `GlassModalBackdrop.tsx` | Blurred glass-effect modal backdrop |
| `MarkdownDisplay.tsx` | Renders markdown content natively |
| `MediaLightbox.tsx` | Full-screen media viewer (pinch-to-zoom images, video player) |
| `SearchBar.tsx` | Text input styled as search bar with icon |
| `SectionHeader.tsx` | Section title with optional "See all" link |
| `StatBox.tsx` | Numeric stat display box (followers, posts count) |
| `Tag.tsx` | Small tag/chip component |
| `VideoThumbnail.tsx` | Video preview thumbnail with play icon overlay |
